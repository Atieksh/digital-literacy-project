## Project Overview

This Digital Literacy Project was completed as part of the CSE0001 course at VIT Bhopal University. The objective of this project is to enhance digital literacy skills, promote cybersecurity awareness, and develop professional communication skills among students.

The project consists of five tasks that focus on different aspects of digital awareness, including infographic design, portfolio creation, coding platform exploration, professional email etiquette, and cybercrime awareness.

---

## Tasks Included

### Task 1 — Digital Literacy Awareness Infographic  
Created an infographic using Canva to spread awareness about digital literacy, safe internet practices, professional online presence, and email etiquette.

### Task 2 — Digital Portfolio Creation  
Created professional accounts on GitHub, LinkedIn, and Kaggle to build a digital portfolio.

### Task 3 — Coding Platform Exploration  
Used HackerRank to practice coding and created a Google Form for digital awareness.

### Task 4 — Professional Email Etiquette  
Drafted professional emails and created a social media etiquette checklist.

### Task 5 — Cybercrime Awareness (Phishing)  
Studied phishing attacks and created prevention strategies.
